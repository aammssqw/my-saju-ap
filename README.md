
# 🔮 딱이만 사주 - 프리미엄 운명 분석 서비스

이 프로젝트는 **React**와 **Google Gemini AI**를 활용하여 제작된 고품격 사주풀이 웹 애플리케이션입니다.

## 🚀 시작하기

1. **저장소 복제**: `git clone https://github.com/사용자이름/my-saju-app.git`
2. **패키지 설치**: `npm install`
3. **환경 변수 설정**: `.env` 파일을 생성하고 `API_KEY=여기에_제미나이_키_입력`을 넣으세요.
4. **실행**: `npm run dev`

## 🛠 기술 스택
- **Frontend**: React 19, TypeScript, Tailwind CSS
- **AI**: Google Gemini 2.0 Flash (via @google/genai)
- **Design**: Premium Black & Gold Theme

## 💳 비즈니스 모델
- 기본 사주 핵심 요약 제공 (무료)
- 상세 재물/애정/비책 풀이는 카카오톡 상담 연동 (유료)
